# Chinese Lorem

Generate lorem ipsum Chinese text.

中文测试文本生成器

## Installation

`npm install chinese-lorem`

## Usage

```javascript
const chineseLorem = require('chinese-lorem');

chineseLorem('10c');
chineseLorem('10w');
chineseLorem(10, 'c');
chineseLorem(10, 'w');

```

## Test

`npm test`
